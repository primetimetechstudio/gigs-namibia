# Gigs Namibia
Gigs Namibia is a web platform where people can post and find gigs, side hustles, and job opportunities in Namibia.
## Features
-Users can post gigs or jobs
-Browse available opportunities
-Simple and fast interface
-Mobile-friendly design
## Built with
-HTML
-CSS
-Javascript
## Project Structure
index.html-Main page
style.css-Styling
script.js-Functionality
## How to Run the Project
1. Download or clone the repository
2. Open `index.html` in your browser
3. Start exploring the site
## Future Improvements
-User login system
-Database for storing gigs
